# Session Workflow

**Purpose**: Execute step guide through interactive implementation and capture progress.

## Implementation Process

### 1. Execute Step Guide
Work through `step-guide.md` systematically, engaging in collaborative dialogue when:
- Clarification needed on requirements or implementation
- Optimization opportunities discovered  
- Problems identified that need resolution
- Creative ideas emerge during implementation

### 2. Generate Checkpoint
Execute: [**Checkpoint Generation**](.prompts/checkpoint-generate.md)
- Capture completed work, architectural decisions, and context
- Document unimplemented ideas and next step preparation needs

**Output**: Updated `checkpoint.md` with current development state
