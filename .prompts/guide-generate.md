# Step Guide Generation

**Purpose**: Create comprehensive implementation instructions for the next development step.

## Instructions

Generate a detailed `step-guide.md` artifact for the **IMMEDIATE NEXT** step tagged in `roadmap.md`, including:

- **Overview**: Step objectives, scope, and time estimate (20-30 minutes)
- **Implementation Requirements**: Technical specifications and architecture details
- **Code Examples**: Pattern-compliant implementation examples following `style-guide.md`
- **Security Considerations**: Input validation, key material protection, and security requirements
- **Testing Requirements**: Comprehensive test specifications with edge cases
- **Verification Steps**: Clear completion criteria and validation procedures

Base the guide on current project context, established patterns, and requirements from the overall project specification.

**Output**: Complete `step-guide.md` artifact ready for session execution
